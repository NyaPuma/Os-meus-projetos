# 📂 Exercícios de Algoritmos

Esta pasta contém exercícios práticos de **lógica e algoritmos** desenvolvidos durante o curso de *Software Developer*.

---

## 📌 Descrição
Aqui encontras exercícios focados em:
- Estruturas de dados (listas, pilhas, filas, etc.).
- Algoritmos de ordenação e pesquisa.
- Resolução de problemas lógicos.

---

## 🛠 Tecnologias
- **Linguagens**: Python, C#
- **Ferramentas**: Visual Studio, VS Code

---

## 📁 Estrutura
Os ficheiros estão organizados por **tema ou exercício**. Cada ficheiro contém:
- O enunciado do problema.
- A solução implementada.
- Comentários explicativos (quando aplicável).
