# 📂 Exercícios de Bases de Dados

Esta pasta contém exercícios práticos de **bases de dados**, incluindo modelação, consultas SQL e design de esquemas.

---

## 📌 Descrição
Aqui encontras exercícios focados em:
- Modelação de bases de dados (diagramas ER).
- Consultas SQL (SELECT, INSERT, UPDATE, DELETE, JOINs, etc.).
- Otimização de bases de dados.

---

## 🛠 Tecnologias
- **Linguagens**: SQL
- **Ferramentas**: MySQL, SQL Server, SQLite, Draw.io (para diagramas)

---

## 📁 Estrutura
Os ficheiros estão organizados por:
- **Diagramas**: Ficheiros `.drawio` ou `.png` com modelos conceituais.
- **Scripts SQL**: Ficheiros `.sql` com consultas e criação de tabelas.
- **Documentação**: Ficheiros `.md` ou `.txt` com explicações.
