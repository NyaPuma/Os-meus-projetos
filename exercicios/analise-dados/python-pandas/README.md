# 📂 Análise de Dados com Python e Pandas

Esta pasta contém projetos de **análise de dados** usando **Python e Pandas**. Os projetos estão em desenvolvimento e focam-se em manipulação, limpeza e visualização de dados.

---

## 📌 Descrição
Aqui encontras:
- Scripts para manipulação de ficheiros CSV.
- Análise exploratória de dados (EDA).
- Visualizações com Matplotlib ou Seaborn.

---

## 🛠 Tecnologias
- **Linguagens**: Python
- **Bibliotecas**: Pandas, NumPy, Matplotlib, Seaborn
- **Ferramentas**: Jupyter Notebook, VS Code

---

## 📁 Estrutura
- **Ficheiros `.ipynb`**: Notebooks Jupyter com código e visualizações.
- **Ficheiros `.py`**: Scripts Python independentes.
- **Ficheiros `.csv`**: Dados de exemplo ou datasets.
