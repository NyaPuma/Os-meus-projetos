# 💻 Projetos em C#

Esta pasta contém projetos desenvolvidos em **C#**, incluindo aplicações de consola, Windows Forms e outros exemplos práticos.

---

## 📌 Descrição
Aqui encontras:
- Aplicações de consola para praticar lógica e sintaxe.
- Projetos Windows Forms para desenvolvimento de interfaces gráficas.
- Exemplos de integração com bases de dados (SQL Server, SQLite).

---

## 🛠 Tecnologias
- **Linguagem**: C#
- **Ferramentas**: Visual Studio, .NET Framework

---

## 📁 Estrutura
- **Aplicações de Consola**: Pastas com projetos `.csproj` e ficheiros `.cs`.
- **Windows Forms**: Pastas com formulários e lógica de UI.
- **Bases de Dados**: Ficheiros `.sql` ou conexões a bases de dados.
