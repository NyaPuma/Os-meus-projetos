# 📂 Os Meus Projetos

Um repositório pessoal para guardar e partilhar mini projetos desenvolvidos em várias áreas: **programação, bases de dados, sistemas embebidos, modelação 3D e web**.

---

## 📌 Sobre
Este repositório serve como portfólio e arquivo de projetos pessoais, criados para aprendizado, experimentação e prática de competências em diferentes tecnologias.
