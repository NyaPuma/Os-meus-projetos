# 🎬 Teste de Animações - Website

Esta pasta contém **testes e experiências com animações web** usando HTML, CSS e JavaScript.

---

## 📌 Descrição
- **Objetivo**: Experimentar técnicas de animação para websites.
- **Tecnologias**: Animações CSS, JavaScript (GSAP, Anime.js, etc.).
- **Resultados**: Exemplos de animações interativas e suaves.

---

## 🛠 Tecnologias
- **Linguagens**: HTML5, CSS3, JavaScript
- **Bibliotecas**: GSAP, Anime.js, CSS Animations
- **Ferramentas**: VS Code, CodePen (para testes rápidos)

---

## 📁 Estrutura
```
teste-animacoes/
├── index.html          # Página com exemplos de animações
├── styles/            # Ficheiros CSS
│   └── animations.css
├── scripts/           # Ficheiros JavaScript
│   └── animations.js
├── images/            # Imagens usadas nas animações
│   └── ...
└── README.md          # Este ficheiro
```

---

## 🚀 Como Executar
1. Abre o ficheiro `index.html` num navegador.
2. Ou, se estiveres a usar um servidor local (ex: Live Server no VS Code):
   - Instala a extensão **Live Server**.
   - Clica em "Go Live" no canto inferior direito do VS Code.

---

## 🎨 Exemplos de Animações
- **CSS Animations**: Animações simples com `@keyframes`.
- **GSAP**: Animações avançadas com GreenSock.
- **Anime.js**: Animações leves e fáceis de implementar.
