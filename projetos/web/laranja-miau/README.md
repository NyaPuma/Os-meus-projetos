# 🍊 Laranja Miau - Website

Website desenvolvido para o projeto **Laranja Miau**, um conceito de loja online ou plataforma de adoção de gatos.

---

## 📌 Descrição
- **Objetivo**: Criar um website atrativo e funcional para o projeto Laranja Miau.
- **Funcionalidades**:
  - Página inicial com informações sobre o projeto.
  - Galeria de imagens (gatos, produtos, etc.).
  - Formulário de contacto.
- **Design**: Protótipo criado no Figma (ver link abaixo).

---

## 🛠 Tecnologias
- **Linguagens**: HTML5, CSS3, JavaScript
- **Ferramentas**: Figma (design), VS Code

---

## 🚀 Como Executar
1. Abre o ficheiro `index.html` num navegador.
2. Ou, se estiveres a usar um servidor local (ex: Live Server no VS Code):
   - Instala a extensão **Live Server**.
   - Clica em "Go Live" no canto inferior direito do VS Code.
