# ⚙️ CNC - Torno

Este projeto consiste em **programas e designs para um torno CNC** (Controle Numérico Computadorizado).

---

## 📌 Descrição
- **Objetivo**: Criar programas para usinagem de peças em um torno CNC.
- **Aplicações**: Fabrico de peças cilíndricas, roscas, e outras geometrias.
- **Resultados**: Ficheiros G-code prontos para serem executados numa máquina CNC.

---

## 🛠 Tecnologias
- **Software**: Mach3, Fusion 360 (para simulação), NCView (para visualização de G-code)
- **Formatos**: `.NC` (G-code), `.TAP`

---

## 📁 Estrutura
- **Programas CNC**: Ficheiros `.NC` ou `.TAP` com o código G-code.
- **Desenhos**: Ficheiros `.DXF` ou `.STEP` das peças a usinar.
- **Simulações**: Ficheiros de simulação (ex: vídeos ou capturas de ecrã).

---

## 🚀 Como Executar
1. Carrega o ficheiro G-code no software da máquina CNC (ex: Mach3).
2. Faz uma simulação para verificar o programa.
3. Executa o programa na máquina.

---

## ⚠️ Segurança
- **Sempre** faz uma simulação antes de executar o programa na máquina.
- Usa **proteção ocular** e equipamento de segurança adequado.
- Certifica-te de que a peça está **corretamente fixa** na máquina.
