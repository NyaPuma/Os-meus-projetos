# 👁️ Visão Computacional - Detecção de Bordas

Este projeto consiste num sistema de **detecção de bordas de objetos usando cores** desenvolvido em **Python**.

---

## 📌 Descrição
- **Objetivo**: Detetar as bordas de objetos em imagens usando técnicas de processamento de imagem baseadas em cores.
- **Aplicações**: Visão industrial, robótica, automação.
- **Resultados**: Algoritmo funcional para detecção de bordas em tempo real ou em imagens estáticas.

---

## 🛠 Tecnologias
- **Linguagem**: Python
- **Bibliotecas**: OpenCV, NumPy
- **Ferramentas**: VS Code, Jupyter Notebook

---

## 📁 Estrutura
- **Código**: Ficheiros `.py` com o algoritmo de detecção de bordas.
- **Imagens de Teste**: Pastas com imagens para testar o algoritmo.
- **Documentação**: Ficheiros `.md` com explicações técnicas.

---

## 🚀 Como Executar
1. Instala as dependências:
   ```bash
   pip install opencv-python numpy
   ```
2. Executa o script principal:
   ```bash
   python deteccao_bordas.py
   ```
