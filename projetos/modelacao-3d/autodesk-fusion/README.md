# 🎨 Geometrias Básicas - Autodesk Fusion 360

Este projeto consiste na modelação de **geometrias básicas** usando **Autodesk Fusion 360**.

---

## 📌 Descrição
- **Objetivo**: Praticar a criação de formas geométricas simples e complexas.
- **Aplicações**: Base para projetos mais avançados de modelação 3D.
- **Resultados**: Modelos 3D de geometrias como cubos, esferas, cilindros, etc.

---

## 🛠 Tecnologias
- **Software**: Autodesk Fusion 360
- **Formatos**: `.F3D` (ficheiros nativos do Fusion 360)

---

## 📁 Estrutura
- **Modelos**: Ficheiros `.F3D` de cada geometria.
- **Exportações**: Ficheiros `.STEP` ou `.STL` para compatibilidade.
