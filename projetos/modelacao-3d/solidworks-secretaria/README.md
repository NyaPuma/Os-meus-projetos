# 🏭 Projeto SolidWorks - Secretaria

Este projeto consiste num modelo 3D de uma **secretaria** desenvolvido em **SolidWorks**.

---

## 📌 Descrição
- **Objetivo**: Criar um modelo 3D funcional de uma secretaria com gavetas e prateleiras.
- **Dificuldades**: Modelação de peças complexas e montagens.
- **Resultados**: Modelo 3D completo e renderizações realistas.

---

## 🛠 Tecnologias
- **Software**: SolidWorks
- **Formatos**: `.SLDPRT` (peças), `.SLDASM` (montagem)

---

## 📁 Estrutura
- **Peças**: Ficheiros `.SLDPRT` das partes individuais (ex: gavetas, pernas, tampo).
- **Montagem**: Ficheiro `.SLDASM` com a secretaria completa.
- **Renderizações**: Imagens `.png` ou `.jpg` do modelo final.
