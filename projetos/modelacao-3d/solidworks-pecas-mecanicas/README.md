# ⚙️ Projeto SolidWorks - Peças Mecânicas

Este projeto consiste num conjunto de **peças mecânicas** modeladas em **SolidWorks**.

---

## 📌 Descrição
- **Objetivo**: Criar peças mecânicas standard (ex: engrenagens, eixos, suportes).
- **Aplicações**: Utilização em montagens maiores ou como exercício de modelação.
- **Resultados**: Peças 3D prontas para impressão ou usinagem.

---

## 🛠 Tecnologias
- **Software**: SolidWorks
- **Formatos**: `.SLDPRT` (peças individuais)

---

## 📁 Estrutura
- **Peças**: Ficheiros `.SLDPRT` de cada peça mecânica.
- **Desenhos Técnicos**: Ficheiros `.SLDDRW` (se disponíveis).
- **Exportações**: Ficheiros `.STEP` ou `.STL` para compatibilidade.
